## 🚀 Project Overview

**QurbaniHat** is a modern livestock booking platform where users can explore, view details, and book animals such as cows and goats for Qurbani.  
The platform includes authentication, dynamic routing, and a responsive UI for smooth user experience.

## ✨ Key Features

- 🐄 Browse livestock animals (cow, goat, etc.)
- 🔍 View detailed animal information
- 🔐 Email & Google authentication system
- ⚡ Fast Next.js App Router architecture
- 📱 Fully responsive UI (mobile + desktop)
- 🎨 Modern design with Tailwind CSS & animations
- 🔄 Dynamic routing for animal details pages
- 🧾 Form validation using HeroUI
- 🔔 Toast notifications for actions
- ☁️ Deployed on Vercel
